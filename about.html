About this project

Unfortunately we don't actually have any monsters to rent, would be pretty cool if we did right but all of our monsters are ours so hands off!

This site is actually part of a blog post written by Aaron Williams, web designer at Code Enigma. The blog was a challenge to design and build a website in a day. You can read the full article here.

There may actually be something here that you would like get the source for, please help yourself but be warned, designing and building a website in a day doesn't make for the cleanest code ;)

A few quick links for things I used while building this site; gulp.js for the build, Gridism for the grid, Hipster Sound for the geometric background and StockSnap for open source imagery.
